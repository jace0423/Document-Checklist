# Document-Checklist

課程內容專業驗收表管理系統

## 📖 專案概述

本專案是一個互動式的**課程內容驗收表單系統**，用於 PM（產品經理）快速填寫和管理課程製作相關的文檔信息。系統支持在線填寫、數據持久化、JSON 匯出/匯入、以及列印功能。

## ✨ 主要功能

### 1. 表單管理
- **課程課綱/腳本連結**：記錄雲端連結和後台連結
- **製作項目信息**：包括課程組合、負責PM、章節名稱、合作講師
- **動態單元表格**：支援新增/刪除課程單元，自動更新項次編號

### 2. 附件清單
- 講稿／逐字稿
- 課程教材／補充資料
- 簡報檔（PPT／PDF）
- 作業
- 其他（自定義描述）

### 3. 數據操作
- ✅ **匯出 JSON**：將表單數據導出為JSON檔案，便於傳遞給 FA 審核人員
- ✅ **匯入 JSON**：從之前導出的JSON檔案快速載入數據
- ✅ **列印表單**：優化的打印樣式，可直接列印
- ✅ **清空表單**：一鍵清空所有數據（需確認）

### 4. 用戶體驗
- 自動調整 Textarea 高度以適應內容
- 響應式設計，支援桌面和平板設備
- 即時成功提示訊息
- 清晰的分節結構和指引

## 🚀 快速開始

### 使用方式

1. **打開表單**
   - 在瀏覽器中打開 `PM_course_inspection_form.html` 檔案

2. **填寫信息**
   - 依序填寫課程相關信息
   - 在表格中新增或刪除課程單元
   - 勾選收到的附件清單

3. **匯出數據**
   - 點擊「📥 匯出 JSON 資料」按鈕
   - 自動下載 JSON 檔案（命名格式：`PM_課程驗收表_[章節名稱]_[日期].json`）
   - 將檔案交給 FA 審核人員

4. **匯入數據**
   - 點擊「📤 匯入 JSON 資料」按鈕
   - 選擇之前導出的 JSON 檔案
   - 表單自動填入相應數據

5. **列印與清空**
   - 「🖨️ 列印表單」：打開瀏覽器列印對話框
   - 「🗑️ 清空表單」：清除所有已填寫內容

## 📋 表單結構

### 一、課程課綱/腳本＆後台連結
- 雲端連結（Google Drive、OneDrive 等）
- 後台連結（課程管理系統）

### 二、製作項目
- 課程組合
- 負責 PM
- 章節名稱
- 合作講師
- 課程單元表（項次、名稱、分鐘數、驗收日期）

### 三、附件清單
- 附件清單勾選
- 上架人員
- 上架日期
